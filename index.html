Die Benützung von iPhone und iPad Applikationen ist einfach (zumindest meistens). Genau das selbe dachte mein Partner und Designer Luis Martins als er auf der Suche war nach einem Slogan für Fidelis Factory. Deshalb entwickeln wir Applikationen für iPhone und iPad die das alltägliche Leben vereinfachen.

Je einfacher die Applikation für den Benutzer, desto schwieriger ist es für die Entwickler trotzdem einen grossen Funktionsumfang anzubieten. Aus diesem Grund stellen wir einige Komponenten zur Entwicklung von Applikationen zur Verfügung, die nicht nur das Leben der Bentuzer, sondern auch das Leben der Programmierer etwas vereinfachen.

Fühlt euch frei den zur Verfügung gestellten Quellcode zu verwenden oder zu verändern. Wir würden uns sehr über ein Feedback oder eine Benachrichtigung freuen, falls Ihr eine unserer Komponenten in eurer Applikation verwendet.

Im Namen des Fidelis Factory Team

SMLoginView

Die SMLoginView bietet eine einfache Möglichkeit, wenn eine Applikation mit einem Passwortschutz versehen werden soll und die dafür notwendige Loginmaske fehlt.

Die View macht die Daten über Delegate Methoden einfach zugänglich.

JSON Webservice Tester

Der JSON Webservice tester ist ein Tool zum testen von JSON webservicen.

Es sendet die eingegebenen Daten über einen HTTP POST request an die angegebene URL und zeigt das Resultat an.

Weitere Module

Wir haben ein ganzes Framework an Modulen die den Aufbau einer Applikation erleichtern. Diese sind leider nicht alle in einem Opensource würdigen Zustand. Zuerst benötigen wir etwas Zeit um diese fertig zu machen.
