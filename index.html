Luis Martins, Co-Founder und Graphics Designer von Fidelis Factory

Die Benutzung von iPhone und iPad Applikationen ist einfach (zumindest meistens). Genau das selbe dachte mein Partner und Designer Luis Martins, als er auf der Suche war nach einem Slogan für FidelisFactory. Denn denau entwickeln wir Applikationen für iPhone und iPad, die das alltägliche Leben vereinfachen.

Je einfacher die Applikation für den Benutzer, desto schwieriger ist es für die Entwickler trotzdem einen grossen Funktionsumfang anzubieten. Aus diesem Grund stellen wir einige Komponenten zur Entwicklung von Applikationen zur Verfügung, die nicht nur das Leben der Benutzer, sondern auch das Leben der Programmierer etwas vereinfachen sollen.

Fühlt euch frei den zur Verfügung gestellten Quellcode zu verwenden oder zu verändern. Wir würden uns sehr über ein Feedback oder eine Benachrichtigung freuen, falls Ihr eine unserer Komponenten in eurer Applikation verwendet.

Im Namen des Fidelis Factory Teams

Sandro Meier

SMLoginView

Die SMLoginView bietet eine einfache Möglichkeit, eine individualisiertbare Loginmaske für eine Applikation zu generieren, um diese mit einem Passwortschutz zu versehen.

Die View macht die Daten über Delegate Methoden einfach zugänglich.

JSON Webservice Tester

Der JSON Webservice tester ist ein Tool zum testen von JSON webservicen.

Es sendet die eingegebenen Daten über einen HTTP POST Request an die angegebene URL und zeigt das Resultat an.

Weitere Module

Wir verfügen über ein ganzes Framework an Modulen, die den Aufbau einer Applikation erleichtern. Diese sind leider nicht alle in einem Opensource-würdigen Zustand. Wir benötigen noch etwas Zeit um diese vollständig fertig zu stellen.
